Integration Guide

Miro recovers abandoned carts by detecting when a customer starts checkout but doesn't finish. To make this work, you need to send checkout data to Miro from your store's checkout page.

There are two ways to integrate:

  1. JavaScript snippet — paste a small script into your checkout page (works with any platform)
  2. Shopify webhook — automatic capture via Shopify's built-in webhook system (zero code)

Both methods require an API key, which you can create from your Dashboard.

Step 1: Create an API Key

  1. Log in to your Miro Dashboard
  2. Scroll to the API Keys section
  3. Click + New Key
  4. Enter your shop domain (e.g. mystore.myshopify.com) and choose Test or Live mode
  5. Copy the key immediately — it's only shown once

Use Test mode while setting up. Test keys work identically but are clearly labeled in your dashboard so you can distinguish test data from real checkouts. Switch to Live when you're ready to go live.

Step 2: Add the Checkout Snippet

Paste the following script into your checkout page, just before the closing </body> tag. It detects when a customer fills in their contact info and sends it to Miro so we can follow up if they don't complete the purchase.

<script>
(function() {
  var MIRO_API_KEY = 'YOUR_API_KEY';   // Replace with your key
  var MIRO_ENDPOINT = 'https://trymiro.onrender.com/capture';

  // Unique token for this checkout session
  var checkoutToken = sessionStorage.getItem('miro_ck')
    || Math.random().toString(36).substr(2) + Date.now().toString(36);
  sessionStorage.setItem('miro_ck', checkoutToken);

  var sent = {};

  function capture(data) {
    // Don't re-send identical data
    var key = JSON.stringify(data);
    if (sent[key]) return;
    sent[key] = true;

    var payload = {
      checkout_token: checkoutToken,
      email: data.email || null,
      phone: data.phone || null,
      cart_value: data.cart_value || null,
      cart_items: data.cart_items || null
    };

    fetch(MIRO_ENDPOINT, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer ' + MIRO_API_KEY
      },
      body: JSON.stringify(payload)
    }).catch(function() {});
  }

  // Auto-capture when email or phone fields lose focus
  document.addEventListener('focusout', function(e) {
    var el = e.target;
    if (!el || !el.value) return;

    var data = {};
    if (el.type === 'email' || el.name === 'email') {
      data.email = el.value;
    } else if (el.type === 'tel' || el.name === 'phone') {
      data.phone = el.value;
    } else {
      return;
    }

    capture(data);
  });

  // Expose globally so you can call it manually
  window.miroCapture = capture;
})();
</script>

That's it. As soon as a customer types their email or phone on your checkout page and moves to the next field, Miro receives the data. If they abandon, our AI agent follows up to recover the sale.

Snippet Reference

The snippet auto-captures email and phone from form fields. You can also call window.miroCapture() manually to send additional data like cart value and items.

Manual Capture Example

// Call this wherever you have cart data available
window.miroCapture({
  email: 'customer@example.com',
  phone: '+15551234567',
  cart_value: 89.99,
  cart_items: [
    { name: 'Classic T-Shirt', qty: 2, price: 29.99 },
    { name: 'Sneakers', qty: 1, price: 30.01 }
  ]
});

*At least one of email or phone is needed for Miro to reach the customer.

How It Works

  1. Customer starts checkout and enters their email/phone
  2. The snippet sends the data to Miro as a "captured" checkout
  3. If the customer completes the purchase, the checkout is marked as converted (via webhook or your own call)
  4. If they abandon, Miro's AI agent calls or texts them to recover the sale

Alternative: Shopify Webhook (Zero Code)

If you're on Shopify, you can skip the JavaScript snippet entirely. Shopify will automatically notify Miro when a checkout is created.

Setup

  1. In your Shopify admin, go to Settings → Notifications → Webhooks
  2. Click Create webhook
  3. Event: Checkout creation
  4. Format: JSON
  5. URL: enter the webhook URL shown below
https://trymiro.onrender.com/webhook

Shopify signs every webhook with an HMAC header. Miro verifies this automatically — no extra setup needed on your end.

Once active, every abandoned checkout on your store is captured automatically. You can see them in your Dashboard within seconds.

Verify Your Integration

After adding the snippet or webhook:

  1. Open your store's checkout page in a new browser tab
  2. Enter a test email (e.g. test@example.com) and move to the next field
  3. Go to your Miro Dashboard
  4. You should see a new checkout with status captured

If you're using a Test mode key, the checkout will appear with a "test" badge in your dashboard. No recovery calls will be made for test checkouts.
